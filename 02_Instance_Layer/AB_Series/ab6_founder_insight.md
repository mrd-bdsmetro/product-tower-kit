## Domain Expertise

### What MR.D Knows That AI/Outsiders Don't

1. **Vietnamese Developer Culture**
   - Vietnamese devs are price-sensitive but value quality
   - Facebook groups are primary community channels
   - "Làm nhanh" (build fast) culture dominates
   - Validation-first mindset is rare but growing

2. **ADHD Founder Reality**
   - ADHD founders start many projects, finish few
   - Clear next actions are critical (not "do research")
   - One-tier-at-a-time prevents overwhelm
   - Dopamine-driven decisions need guardrails

3. **PM in Vietnam**
   - PM role is nascent in Vietnam
   - Most developers are "full-stack" (do everything)
   - Structured PM is seen as "enterprise" (not for solo)
   - Opportunity: Make PM accessible to solo founders

4. **Claude Code Ecosystem**
   - Claude Code is gaining traction among power users
   - Skills system is the right integration point
   - Vietnamese-first is a blue ocean
   - Community building before monetization
