## Selected Target: Vibe Coders + Startup Founders (PRIMARY)

**Why this segment:**
1. **Market size** - 3M+ vibe coders, 5M+ founders = huge market
2. **Growing fast** - Vibe coding is exploding 10x in 2025
3. **Underserved** - No PM framework for vibe coders exists
4. **Willingness to pay** - Devs buy tools ($49-199 range)
5. **Recurring value** - Use for every project

- **PM Freelancers** - Use framework for clients
