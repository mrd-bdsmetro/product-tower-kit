# T2: Market Segmentation - Product Tower Kit (UPDATED)

## Segments Identified (CORRECTED - Broader)

### Segment 1: Vibe Coders (PRIMARY)
- **Demographic:** 20-45, global (US, EU, Asia), tech-savvy
- **Behavior:** Uses AI coding tools daily, ships fast, values workflow
- **Psychographic:** Early adopter, productivity-focused, willing to try new tools
- **Size:** 3,000,000+ globally 📊
- **WTP:** $49-199 one-time
- **Growth:** 10x in 2025 📊

### Segment 2: Startup Founders (PRIMARY)
- **Demographic:** 25-45, global, ambitious
- **Behavior:** Validates ideas, ships MVPs, iterates fast
- **Psychographic:** Anti-corporate, values autonomy, learns by doing
- **Size:** 5,000,000+ globally 📊
- **WTP:** $49-199 one-time

### Segment 3: Vietnamese Solo Founders (SECONDARY)

- **Behavior:** Uses AI coding tools daily, builds side projects
- **Psychographic:** Ambitious, time-poor, values efficiency
- **Size:** 50,000+ 📊
- **WTP:** $49-99 one-time

### Segment 4: PM Freelancers (SECONDARY)
- **Demographic:** 25-45, global, experienced PMs
- **Behavior:** Uses frameworks for client work
- **Psychographic:** Professional, values structure
- **Size:** 500,000+ globally 📊
- **WTP:** $99-199 one-time

### Segment 5: Agencies/Dev Shops (TERTIARY)
- **Demographic:** 2-50 person teams, global
- **Behavior:** Uses templates for client projects
- **Psychographic:** Efficiency-focused, values reusability
- **Size:** 100,000+ globally 📊
- **WTP:** $199+ one-time

### Segment 6: Indie Hackers (TERTIARY)
- **Demographic:** 20-40, global, bootstrappers
- **Behavior:** Ships fast, validates with landing pages
- **Psychographic:** Anti-corporate, values autonomy
- **Size:** 500,000+ globally 📊

### Segment 7: Students/Bootcamp (FUTURE)
- **Demographic:** 18-30, global, learning
- **Behavior:** Learning PM, building projects

- **Size:** 1,000,000+ globally 📊
