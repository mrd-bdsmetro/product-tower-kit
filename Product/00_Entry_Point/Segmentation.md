# Segmentation

## Market Segments

### Segment 1: Solo Founder (Primary)
| Attribute | Value |
|-----------|-------|
| **Profile** | One-person business, side project |
| **Age** | 28-40 |
| **Tech skill** | Medium to high |
| **Time available** | Evenings, weekends |
| **Budget** | Limited ($0-100/mo) |
| **Primary need** | Structure, accountability |
| **PMF_impact** | High |

**Behaviors:**
- Searches YouTube/LinkedIn for "how to validate idea"
- Joins indie hacker communities
- Buys courses but doesn't finish them
- Needs external accountability

### Segment 2: Early-Stage Startup (Secondary)
| Attribute | Value |
|-----------|-------|
| **Profile** | 2-3 co-founders, pre-revenue |
| **Stage** | Idea to MVP |
| **Tech skill** | High (technical co-founder) |
| **Time available** | Limited |
| **Budget** | $100-500/mo |
| **Primary need** | Methodology, investor-ready docs |
| **PMF_impact** | High |

**Behaviors:**
- Researches YC, a16z, First Round
- Needs deck-ready frameworks
- Uses Notion for documentation
- Values credibility and structure

### Segment 3: Indie Hacker (Tertiary)
| Attribute | Value |
|-----------|-------|
| **Profile** | Digital nomad, remote work |
| **Age** | 25-35 |
| **Tech skill** | Very high |
| **Time available** | Flexible |
| **Budget** | $0-50/mo |
| **Primary need** | Speed, CLI tools, automation |
| **PMF_impact** | Medium |

**Behaviors:**
- Uses CLI tools heavily
- Prefers open-source
- Values control and flexibility
- Comfortable with templates

### Segment 4: ADHD Brain (Specialty)
| Attribute | Value |
|-----------|-------|
| **Profile** | Has ADHD diagnosis or symptoms |
| **Core need** | Low friction, short cycles, rewards |
| **Pain point** | Starting, focusing, following through |
| **PMF_impact** | Very High |

**Behaviors:**
- Needs immediate feedback
- Distracted by complexity
- Needs clear next action
- Benefits from gamification

## Segment Prioritization

| Priority | Segment | Rationale |
|----------|---------|-----------|
| 1 | Solo Founder + ADHD | Core user, highest PMF impact |
| 2 | Solo Founder (general) | Similar needs, larger volume |
| 3 | Early-Stage Startup | Higher budget, credibility |
| 4 | Indie Hacker | Lower priority, price sensitive |

## Segment-Specific Adaptations

### For Solo Founder + ADHD
- Quick_Start_Guide with 15-min max
- Visual progress tracking
- Short dopamine hits (checkmarks, scores)
- Avoid walls of text

### For Early-Stage Startup
- Investor-ready outputs
- Notion-compatible templates
- Team collaboration features
- ROI calculator

### For Indie Hacker
- CLI-first tooling
- Git-based workflow
- Open-source friendly
- Minimal dependencies
