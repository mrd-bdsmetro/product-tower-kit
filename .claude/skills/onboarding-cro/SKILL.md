# Onboarding CRO — User Activation

## Goal
Optimize post-signup onboarding to reach "aha moment" faster.

---

## Defining Activation

### Aha Moment
The moment when users first experience the core value of your product.

**Examples:**
- Slack: "10,000 messages" team engagement
- Dropbox: "1 file shared" collaboration
- Facebook: "10 friends" network effect
- Product Tower: "T0 research complete" validation

### Activation Metric
| Product | Activation Event | Target Time |
|---------|-----------------|-------------|
| SaaS | First value delivered | < 5 minutes |
| Tool | First output created | < 2 minutes |
| Framework | First tier completed | < 30 minutes |

---

## Onboarding Flow Design

### 1. Welcome Screen
- Clear value proposition
- What to expect
- Single CTA

### 2. Setup Checklist
- 3-5 steps max
- Progress indicator
- Skip option

### 3. First Action
- Guided experience
- Tooltip/hint
- Quick win

### 4. Empty State
- Show what's possible
- Provide templates
- Encourage action

### 5. Progress Indicators
- Checklist completion
- Milestone badges
- Progress bar

---

## Multi-Channel Onboarding

| Channel | Timing | Content |
|---------|--------|---------|
| **In-app** | Immediate | Tooltips, modals |
| **Email** | Day 1, 3, 7 | Tips, tutorials |
| **Push** | Day 1, 3 | Reminders |
| **SMS** | Day 1 | Welcome |
| **In-person** | Week 1 | Demo, training |

---

## Handling Stalled Users

| Stall Point | Intervention |
|-------------|--------------|
| Signed up, didn't start | Email: "Need help getting started?" |
| Started, didn't finish | Email: "Pick up where you left off" |
| Finished setup, no usage | Email: "Here's what you can do" |
| Used once, didn't return | Email: "We miss you" |

---

## Measurement

| Metric | Formula | Target |
|--------|---------|--------|
| Activation rate | Activated / Signed up | > 40% |
| Time to value | Time to first value | < 5 min |
| Onboarding completion | Completed / Started | > 60% |
| Day 1 retention | Returned day 1 / Signed up | > 50% |
| Day 7 retention | Returned day 7 / Signed up | > 30% |

---

## Experiment Ideas

1. **Simplify flow** — Remove unnecessary steps
2. **Add guided tour** — Step-by-step walkthrough
3. **Personalize** — Ask goals, customize experience
4. **Quick wins** — Pre-fill templates, show examples
5. **Social proof** — Show what others achieved
6. **Progress** — Show completion percentage
7. **Rewards** — Celebrate milestones
8. **Email sequences** — Drip content over time
9. **In-app messages** — Contextual help
10. **Live chat** — Real-time support
