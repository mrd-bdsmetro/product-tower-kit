# Product x Sale — Unified Framework

## Goal
Merge Product Tower (what to build) + Sales Tower (how to sell) into unified pipeline.

---

## 5 PHASES

| Phase | Name | BUILD Track | SELL Track |
|-------|------|-------------|------------|
| 0 | Intel | Market research | Demand check |
| 1 | Validate | PMF validation | WTP validation |
| 2 | Dual Track | Features + Build | Offer + Lead Gen |
| 3 | GTM | Launch | Sales system |
| 4 | Measure | Analytics | Revenue metrics |

---

## 3 MODES

| Mode | Khi nào | Phases |
|------|----------|--------|
| BUILD ONLY | Product (SaaS, app) | 0→1→2A→3A→4 |
| SELL ONLY | Service (consulting) | 0→1→2B→3B→4 |
| BUILD+SELL | Product + service layer | 0→1→2A+2B→3→4 |

---

## SYNERGY

- Build tools → sell bonuses
- Revenue funds dev
- User feedback → both tracks
