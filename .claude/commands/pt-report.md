# /pt:report — Generate Full Report

## Usage
```
/pt:report
/pt:report [project]
```

## Behavior
1. Read all `data/t*.md` files
2. Generate comprehensive report
3. Include all tiers with status
4. Include PMF analysis
5. Include recommendations

## Output
Generates `data/product_tower_report.md` with:
- Executive summary
- Tier-by-tier analysis
- PMF assessment
- Risk analysis
- Recommendations
- Next steps
